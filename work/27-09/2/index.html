$ bem make

Первая сборка с bem tools займёт некоторое время, так как именно в этот момент в директорию проекта устанавливаются все необходимые npm-пакеты.

Для запуска сервера выполняем:

$ bem server

В результате вы увидете следующие сообщение

info: Server is listening on PORT 8888. Point your browser on xxxxxxxxxxxxxxx

Проблема?

Если порт уже используется другой программой, то -p
